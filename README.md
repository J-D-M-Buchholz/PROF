# Projekt: Virtueller Professor mit OpenAI Text-DaVinci-003 KI

Dieses Projekt wurde mit dem Ziel entwickelt, meine Fähigkeiten in React, API-Integration und der Arbeit mit KI-Systemen zu vertiefen.

## Ziel der Anwendung

Die Anwendung dient als virtueller Professor, um bei Fragen und Problemen zu helfen und das Lernen zu unterstützen. Sie verwendet die Text-Davinci-003-KI von OpenAI und kann auf die meisten Fragen eine Antwort geben.

## Sprachunterstützung

Die Anwendung kann in 12 verschiedenen Sprachen antworten, unabhängig von der Sprache der Anfrage.

## Anpassbare Parameter

Die Antwortlänge kann über Token und die Kreativität über die Temperatur eingestellt werden.

## Modi

Die Anwendung verfügt über zwei Modi:

1. **Frage-Antwort-Modus**: Hier können Sie Fragen stellen und ein Gespräch mit der KI führen.
2. **Code-Überprüfungs-Modus**: Hier können Sie Ihren Code testen lassen. Die Funktionalität wird erklärt und gegebenenfalls werden Fehler aufgezeigt.

## Fazit

Durch die alleinige Programmierung dieser Anwendung konnte ich meine Fähigkeiten in React und API-Integration erweitern und wertvolle Erfahrungen im Umgang mit KI-Systemen sammeln. Die Anwendung bietet ein hervorragendes Beispiel für meine Fähigkeit, komplexe Technologien zu verstehen und in praktische Anwendungen umzusetzen.
